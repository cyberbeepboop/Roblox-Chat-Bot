
# Roblox Chat Bot
FINALLY, a  bot that actually works in roblox... And can work in any other games too, to my knowledge.
This uses pydirectinput.


# Details
Pretty customizable

To install, use:
1. Go to **Python.org**
2. If you have python installed, run python, python3, or py in any terminal
3. Once used, use py -m ensurepip --upgrade in any terminal. (if not found, use python or python3.)
4. Once you have pip installed, run pip install pydirectinput pygetwindow pyperclip pyautogui pynput
5. Run the script, or customize in settings.txt
